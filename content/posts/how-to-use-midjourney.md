Midjourney makes the most striking AI images, but new users often get mediocre results because they don't know how to prompt it. Here's how to get started and get good output fast.

## Step 1: Get access

Subscribe to a [Midjourney plan](/posts/midjourney-pricing) (it usually requires a paid subscription), then use the **web app** or **Discord**. The web app is the easiest starting point.

## Step 2: Write your first prompt

Type a prompt describing the image. Midjourney returns a grid of four options. Then **upscale** the one you like or generate **variations**.

## Step 3: Prompt like a pro

Vague prompts get vague images. Be specific about:

- **Subject** — what's in the image
- **Style** — photoreal, illustration, 3D, a named art style
- **Lighting & mood** — golden hour, moody, vibrant
- **Composition** — close-up, wide shot, top-down

## Step 4: Use parameters

Add parameters to control output, e.g. **aspect ratio** (`--ar 16:9`) and stylization. A little goes a long way — change one thing at a time so you learn what each does.

## Step 5: Iterate and edit

Treat the first grid as a draft. Re-roll, vary, upscale and use the editing tools to refine. The fastest way to improve is to **study prompts behind images you admire** and copy the structure.

## Beginner tips

- Start **simple**, then add detail
- Change **one variable at a time**
- Keep a note of prompts that worked
- For images with **text**, Midjourney struggles — use [Ideogram](/posts/ideogram-review) instead

## Bottom line

Get access, prompt with specific subject/style/lighting/composition, use parameters, and iterate on the grid. Midjourney rewards good prompting more than any other tool — start simple and learn from prompts you admire.
